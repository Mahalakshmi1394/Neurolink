# NeuroLink 🧠

A Unified Digital Health Record System with AI-Based Pre-Symptomatic Neurological Risk Prediction.

## Project Structure

- **client/**: Frontend (React + Vite + Tailwind)
- **server/**: Backend (Node + Express + MongoDB)
- **ai-services/**: AI Modules (Python - Phase 2)

## Setup & Run

### 1. Backend
```bash
cd server
npm install
npm run dev
```

### 2. Frontend
```bash
cd client
npm install
npm run dev
```

## Features
- Unique Health ID
- Doctor & Patient Dashboards
- Secure Medical Records
